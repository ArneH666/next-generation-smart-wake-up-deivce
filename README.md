# next-generation-smart-wake-up-deivce
> An alarm clock that is programmed for an eps32.  
> Short: NgSWD

## Components
- ESP32 Dev Module
- TFT Display
- Temperature Sensor
- Ultrasonic Distance Sensor
- Breadboard
- USB-Port for Power
- Speaker

## Requirements

- [ ] Display time as digital
- [ ] Alarm function with sound / melodie
- [ ] Display room temperature
- [ ] Show day / night
- [ ] Show Moisture
- [ ] Use Ultrasonic sensor to "wave off" the alarm
- [ ] Vehicle base Scania S A4x2
- [ ] Pen Holder
